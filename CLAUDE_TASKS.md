# Claude 修改任務清單 (精簡版)
請針對以下系統進行 Bug 修復與功能優化，視覺維持日系手繪簡約風格與奶茶色調。

### M.Y 小百科
**修改檔案**：`https://raw.githubusercontent.com/yamap610/mywiki/refs/heads/main/index.html`
* **修復**：底部導航列消失、項目詳細資訊點擊無反應。
* **圖片**：修復連結預覽圖無法抓取；新增「點擊照片可放大瀏覽」功能（需與版面和諧）。
* **UI 調整**：分類管理的「垃圾桶」圖示改為「x」；「找不到符合的收藏」文字置中。
* **功能新增**：新增列表「批量選取」功能（支援刪除與修改）。
* **設定頁**：將「備份」功能移入設定頁；設定頁需加上「確定」與「取消」按鈕。
* **版面修正**：檢查所有頁面 CSS，在底部導航欄 (Tab Bar) 上方預留足夠空間 (Padding/Margin)，確保內容滑到底部時絕對不會被遮擋。
